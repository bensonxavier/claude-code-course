# Task 1: Install, verify, and try the CLI

## Objective
Build confidence using Claude Code from the terminal.

## Steps
1. Install Node.js if needed and confirm the version with `node --version`.
2. Install Claude Code with `npm install -g @anthropic-ai/claude-code`.
3. Set your `ANTHROPIC_API_KEY` in your shell profile and reload the shell.
4. Verify the installation with `claude --version` and `claude "say hello"`.
5. Create a tiny sample file and ask Claude to summarize it with file context.
6. Start the interactive REPL and ask two follow-up questions in the same session.

## Deliverable
A short note in [resources/my-notes.md](../../resources/my-notes.md) describing what worked, what confused you, and one thing you want to try next.
