# Week 1-2 Tasks

Use these task files in order to build practical CLI skills with Claude Code.

## Suggested flow

1. Start with [01-main.md](01-main.md) and complete the setup and verification steps.
2. Practice one-shot prompts, REPL interaction, and file-based context.
3. Record your observations in [resources/my-notes.md](../../resources/my-notes.md).

## Completion check

You are done when you can:
- install and verify Claude Code,
- run one-shot prompts and interactive sessions,
- and explain what changed after asking Claude to create or modify a small file.
