# Week 7-8 Tasks

These tasks help you learn MCP concepts by connecting tools and observing how an agent uses them.

## Suggested flow

1. Start with [01-main.md](01-main.md).
2. Configure a simple MCP server.
3. Test one tool with a small prompt and then extend the workflow.

## Completion check

You are done when you can:
- configure a basic MCP server,
- connect a tool to Claude Code or a compatible client,
- and explain how the tool improved the workflow.
