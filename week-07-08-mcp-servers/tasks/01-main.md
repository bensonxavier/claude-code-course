# Task 1: Connect a simple MCP tool

## Objective
Learn how MCP servers expose tools that an agent can use.

## Steps
1. Choose a simple tool such as a filesystem or local database tool.
2. Configure an MCP server that exposes that tool.
3. Connect it to Claude Code or a compatible client.
4. Ask the agent to use the tool for a small task such as listing files or reading a local file.
5. Observe whether the tool call is successful and note any configuration issues.

## Deliverable
A short reflection in [resources/my-notes.md](../../resources/my-notes.md) describing the tool, the setup steps, and the result.
