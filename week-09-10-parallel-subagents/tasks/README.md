# Week 9-10 Tasks

These tasks help you practice orchestrating parallel work across multiple agents or workstreams.

## Suggested flow

1. Start with [01-main.md](01-main.md).
2. Split one problem into several independent parts.
3. Merge the results and review the final outcome.

## Completion check

You are done when you can:
- break a task into parallel workstreams,
- coordinate the work clearly,
- and compare the merged result to a single-agent approach.
