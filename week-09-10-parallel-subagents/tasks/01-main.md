# Task 1: Split a task into parallel workstreams

## Objective
Learn how to coordinate work across multiple agents or sub-tasks.

## Steps
1. Pick a task that can be separated into independent parts.
2. Define a role or objective for each workstream.
3. Ask each workstream to produce a partial result.
4. Merge the results and identify any overlap or conflicts.
5. Note where parallel work was helpful and where coordination was still needed.

## Deliverable
A short summary in [resources/my-notes.md](../../resources/my-notes.md) explaining the division of work and the outcome.
