# Week 5-6 Tasks

These tasks help you practice planning, approval, and loop-based workflows in a structured way.

## Suggested flow

1. Start with [01-main.md](01-main.md).
2. Try the task in plan mode and then in a loop-based workflow.
3. Reflect on what changed between the two approaches.

## Completion check

You are done when you can:
- run a multi-step task in plan mode,
- review and approve or adjust the plan,
- and see how an agentic loop handles iteration.
