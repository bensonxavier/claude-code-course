# Task 1: Practice planning and loop execution

## Objective
Learn how to guide Claude Code through a multi-step task with review and iteration.

## Steps
1. Choose a multi-step task such as refactoring a small script, fixing several issues, or adding a feature.
2. Run the task in plan mode and review the proposed steps carefully.
3. Approve or adjust the plan before execution.
4. Observe how Claude proceeds and note where it needs supervision.
5. Run a second iteration where you give a clearer success criterion and let Claude continue until it is met.

## Deliverable
A brief note in [resources/my-notes.md](../../resources/my-notes.md) describing what plan mode improved and where manual guidance was still needed.
