# Week 3-4 Tasks

These tasks help you practice writing CLAUDE.md files that give Claude Code useful project context.

## Suggested flow

1. Start with [01-main.md](01-main.md).
2. Create a small project folder and add a CLAUDE.md file.
3. Refine the instructions and test how Claude responds to them.

## Completion check

You are done when you can:
- create a project-level CLAUDE.md,
- add rules and project context,
- and explain how the file changed Claude's behavior.
