# Week 11-12 Tasks

These tasks help you practice using Claude Code in a git-based development workflow.

## Suggested flow

1. Start with [01-main.md](01-main.md).
2. Use Claude Code to review changes and suggest commit or PR text.
3. Practice a conflict-resolution workflow in a safe test repository.

## Completion check

You are done when you can:
- generate useful commit messages,
- review diffs with assistance,
- and help resolve simple merge conflicts.
