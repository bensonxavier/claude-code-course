# Task 1: Practice git-assisted review and commit workflows

## Objective
Learn how to use Claude Code as a helper in Git-based development.

## Steps
1. Create or open a small repository with a few tracked changes.
2. Ask Claude to suggest a conventional commit message for the staged changes.
3. Review the diff together and ask for a PR or changelog summary.
4. Create a small merge conflict in a test branch and ask Claude to help resolve it.
5. Compare the result with your own manual review.

## Deliverable
A short note in [resources/my-notes.md](../../resources/my-notes.md) describing which Git workflow felt most useful and why.
