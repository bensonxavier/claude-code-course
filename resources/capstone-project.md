# Capstone Project

The capstone is the culminating exercise for the course. It should bring together the main themes of the program: setup, prompt design, project context, iterative improvement, tool integration, and Git-based review.

## Suggested capstone idea

Create a small “developer helper” workflow in a sample project.

### Requirements
- Use a project-specific CLAUDE.md file.
- Start with a realistic coding task.
- Use planning or loop-based workflows to improve the result.
- Optionally connect one MCP tool or use a Git-based workflow for review.
- Finish by documenting what improved between the first and final attempt.

## What success looks like

The capstone is successful when the final result is genuinely useful and the learner can explain:
- what prompt and context made the biggest difference,
- what failed in the first attempt,
- and how they improved the workflow.
