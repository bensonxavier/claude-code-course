## How to Use This Project
1. Open the current module folder (e.g., `week-01-02-claude-cli/`)
2. Read the module's `README.md` for context
3. Follow the numbered task files in that module's `tasks/` folder
4. Check off completed tasks in `PROGRESS.md`
5. Move to the next module when all tasks are done

## Ground Rules
- Don't skip ahead — each module builds on the last
- Write notes in `resources/my-notes.md` as you go
- If stuck, check `resources/hints.md` before looking up answers
- Commit progress after each task batch
